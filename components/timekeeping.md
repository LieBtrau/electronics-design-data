# RTC
## Characteristics
* Package : SOIC-8 (smaller alternative might be TSSOP-8)
  * pin 3 : /INT
  * pin 7 : FOUT
* Interface : I²C
* Voltage : 1.8V to 5.5V
* Current - Timekeeping : <1.0µA
